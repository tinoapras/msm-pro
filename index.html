Supabase — Backend Database ○ Belum Terhubung
Supabase Dashboard → Settings → API → Project API keys → anon public
Setup Supabase (gratis):
1. Buka supabase.com → Sign Up → New Project
2. Tunggu project ready (~2 menit), buka SQL Editor
3. Copy-paste isi file supabase_schema.sql → klik Run
4. Buka Settings → API → copy Project URL dan anon public key
5. Paste di atas → Test Koneksi → Pull dari Supabase
☁ Google Sheets — Backend Tidak Terhubung
Cara setup:
1. Buka script.google.com → New Project
2. Copy-paste kode dari file MSM_Backend.gs yang sudah didownload
3. Deploy → New deployment → Web App → Execute as: Me → Who has access: Anyone
4. Copy URL deployment → paste di field di atas → Test Koneksi
